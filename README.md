# XadrezWithIA
Xadrez em Java com IA simples

#Explicação
 > Xadrez desenvolvido para a disciplina de POO1
 
 > Proposta no arquivo proposta.pdf
 
 #Compilar
 > Navegar até a pasta onde tem os aquivos
 
 > Executar: 
 ```
  javac Main.java
 ```
 
 #Executar
 > Na pasta em que foi compilado
 
 > Executar
 ````
 java Main
```
